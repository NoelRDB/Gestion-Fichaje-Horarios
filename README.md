# Gestión de Fichaje y Horarios

Este proyecto es una aplicación desarrollada para gestionar de forma eficiente los fichajes de entrada y salida de trabajadores, permitiendo controlar horarios, calcular tiempos trabajados y facilitar la organización de turnos dentro de un entorno laboral, especialmente en sectores como la hostelería donde los horarios suelen ser variables.

La herramienta permite:

- **Registrar fichajes de entrada, salida y descansos** de cada trabajador.
- **Consultar historiales de fichaje en tiempo real** de manera sencilla e intuitiva.
- Integrarse fácilmente como sistema interno de control horario.

El objetivo principal es ofrecer una solución **rápida, accesible y fácil de usar** tanto para empleados como para responsables, cumpliendo con las obligaciones laborales y facilitando la gestión diaria del personal.
